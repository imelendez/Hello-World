County Food Venue Health Inspection Logger

Senior Design

Ismael Melendez | github | linkedin

Problem Statement & Description

In order to support efforts of government to become more transparent and increase access to information. Currently online access to health reports is not available online, access would make restaurants more accountable to patrons.
I will make an web application that facilitates access by the public to current status/report of their favorite restaurants. In addition I will make a simple android app that can demonstrate how the Hidalgo Health department could utilize technology to collect reports quicker and more efficiently from employees out in the field via a android tablet or phone.
This a very practical approach with real-life implementation.

Project Goals and Objectives/Deliverables

Mobile/Tablet Interfaces (updates databases with new information)

Mobile Application needs to provide an interface for employees of the County Health Department will be

  • able to input their daily reports
  • must include timestamp
  • updates by text only
  • images (will be attempt)
  • possibly geolocation at time of input

Web Application needs to provide a public interface for the public to be able to search and view reports.

  • searchable
  • fast
  • incldues restaurant informaiton
  • mobile friendly
  • no login will be require to view inforaiton
